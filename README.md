**<h2>Setup the project</h2>**

  Clone this project in the folder of your choice

  ```bash
   git clone https://github.com/Ryusse/manage-landing-page-master.git
  ```

  Install the dependencies

  If you use npm 

  ```bash
  npm install
  ```

  If you use yarn

  ```bash
  yarn install
  ```

  Run the server

   ```bash
  gulp
  ```
